# GPT Pro Signal Bundle Review Prompt

Read the HMM information-exchange bundle, `signal_snapshot_lite.csv`, and the watchlists.

Constraints:

- Separate model evidence, external public information, and your own inference.
- Use web/search to check recent public explanations for highlighted sectors.
- State uncertainty and identify which claims are model-derived versus externally observed.
- Do not output trading, sizing, buy/sell, recommendation, execution, or portfolio-action advice.
- Produce manual-review questions for the human reviewer.

Suggested response structure:

1. Data freshness and provenance check.
2. Baseline-first risk summary.
3. High baseline-risk sectors and possible public context.
4. Model-baseline conflicts worth manual review.
5. HSMM lifecycle observations.
6. Stage03V probability/readiness limitations.
7. External context to verify with public sources.
8. Manual-review questions.
